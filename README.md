# Gold Price Scraper | برنامج تتبع أسعار الذهب

## 📝 Description / الوصف
**EN:** A professional Python script that scrapes real-time gold prices and saves them into a CSV file for historical tracking.
**AR:** سكربت بايثون احترافي يقوم بسحب أسعار الذهب الحية من الإنترنت وحفظها في ملف CSV لتتبع سجل الأسعار.

## 🚀 Features / المميزات
* **Real-time Scraping:** Gets the latest prices automatically. [سحب الأسعار اللحظية تلقائياً]
* **Data Logging:** Saves date, time, and price in a structured format. [حفظ التاريخ والوقت والسعر بشكل منظم]
* **Error Handling:** Robust code that manages connection issues. [معالجة ذكية لأخطاء الاتصال]

## 🛠️ Tech Stack / التقنيات المستخدمة
* Python 3.x
* BeautifulSoup4
* Pandas
* Requests
